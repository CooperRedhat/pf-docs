Use `pf-docs` to proof and run documentation markdown files, and examples.

## Usage

### Install

NPM install to your repository. Add `pf-docs` to `package.json`.

  ```shell
    $ npm i pf-docs
  ```
or Yarn

  ```shell
    $ yarn add pf-docs
  ```
